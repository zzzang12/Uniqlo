# Uniqlo
유니클로 UT 크롤링 프로젝트

## 배운 것
* http GET 요청 보내기
* goquery로 html에서 원하는 node 가져오기
* css selector
* directory 만들기
* file 만들기
* image download
* string parsing

## 사용법
```shell
git clone github.com/zzzang12/uniqlo.git
go build uniqlo.go
./uniqlo.exe
```
